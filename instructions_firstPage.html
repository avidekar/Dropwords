Welcome to DropWords!

DropWords is a time based word-finding puzzle that combines some elements of Scrabble and Boggle. The game is played on a square board of ( 5x5 ) of letter tiles. With these tiles, you can form words from any adjoining letters, including diagonals. The game is also played with or without the clock.


How to play ?

When selecting a word, click individual letters in sequence. Every time you select a button, the string is checked if it is present in the dictionary of about 180,000 words! Once you click a button, you cannot select the same letter to complete a word! Each word is highlighted when clicked.


Reshuffle Option

If you find yourself a bit stuck, you have the Reshuffle option! With Reshuffle, old set of letters are substituted by the new ones. You can use the Reshuffle button, as many times as you want to!


Clear Option

Everytime you want to clear the selected words, you can do that using the Clear Option and start looking for a new word again!


Scoring System

Since, to complete the first level, you need to score a total of 25 points. If you score more than that, the additional points are added to your score as a bonus for the next level!


Game Levels

There are 7 levels. At the first Level, you have to score 25 points in 60 seconds. With every level, you get an additional 30 seconds to reach your score. This means, the second level will be for 90 seconds and so on.
